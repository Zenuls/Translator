# Транслятор C++ в Python

Проект представляет собой транслятор кода с языка C++ на Python с графическим интерфейсом пользователя.

## Возможности

- **Лексический анализ** C++ кода с поддержкой основных конструкций
- **Синтаксический анализ** и построение абстрактного синтаксического дерева (AST)
- **Генерация Python кода** на основе AST
- **Графический интерфейс** с подсветкой синтаксиса
- **Поддержка основных конструкций C++**:
  - Переменные и массивы
  - Функции и параметры
  - Условные операторы (if/else)
  - Циклы (for, while, do-while)
  - Ввод/вывод (cin/cout)
  - Арифметические и логические операции
  - Тернарный оператор

## Структура проекта

Translator-main:
- main.py = Точка входа
- frontend.py = Графический интерфейс
- lexer.py = Лексический анализатор
- parser.py = Синтаксический анализатор
- ast_nodes.py = Определение узлов AST
- code_generator.py = Генератор Python кода
- syntax_highlighter.py = Подсветка синтаксиса
- translator.py = Основной модуль транслятора


## Установка и запуск

### Требования
- Python 3.8+
- Библиотеки: `customtkinter`

### Установка зависимостей
```
pip install customtkinter
```

### Запуск программы
```
python main.py
```

## Использование

1. Запустите приложение через `main.py`
2. Введите код C++ в левое текстовое поле
3. Нажмите кнопку "Перевести" для генерации Python кода
4. Просмотрите результат в правом текстовом поле
5. Используйте системные сообщения для отладки процесса трансляции

## Пример трансляции

### Входной код (C++):
```
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) {
        cout << "Hello, World!" << endl;
    }
    return 0;
}
```
### Выходной код (Python):
```
def main():
    n = int(input())
    i = 0
    while i < n:
        print('Hello, World!')
        i += 1
    return 0

if __name__ == "__main__":
    main()
```
